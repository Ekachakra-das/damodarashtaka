Damodarāṣṭakam

Web App for the Damodarāṣṭakam prayer.

Choose Your Reading Experience:

  • Detailed word-by-word mode: Sanskrit translation for deep understanding
  • Standard mode: with Sanskrit transliteration and full translation

Use menu button "☰" to switch language or reading mode.

Additional Features:

  • Font Controls: Adjustable text size for comfortable reading
  • Multilingual Support: English, Russian, and Italian translations
  • Dark mode: Toggle for comfortable night reading
  • Offline Mode: Works without internet connection

📱 ⚡ Install as native app (works offline)

Open the app in your browser: Damodarāṣṭakam

  1. On Android/Chromium: tap the browser menu "..." → "Add to Home screen" → "Install app"
  2. On iOS: Safari or Chrome: tap the Share button → "Add to Home Screen"
  3. Desktop Chrome: Use the install button in the address bar or in the menu find → Install
